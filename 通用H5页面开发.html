    # 通用H5页面开发

    # 环境准备

    • 全局安装hekr-cli工具
    npm set registry http://npm.hekr.me
npm i -g @hekr/hekr-cli
    • 使用hekr-cli初始化项目
    # 初始化项目根据需要选择对应的功能
hekr init vue my-project

cd my-project

# 安装依赖
npm i
# 运行调试
npm run dev
    • 进入src文件夹下进行开发
    • 打包项目
    # 运行后会在dist文件夹下生成打包后的代码
npm run build

    • 上传zip包到服务器

      1. 使用console账号登录到console云平台，找到对应产品，然后上传到App界面
      2. 也可以直接采用hekr-cli在命令行上传zip包到云端
      # 运行命令选择对应的账号，然后选择文件上传
hekr page
